关于使用hexo为GitHub个人博客润色

关于hexo

hexo官网

现在的静态博客框架有很多，例如Jekyll、Simple、Pelican等。它们各有利弊，但本篇只介绍hexo静态博客框架。

hexo基于Node.js，支持markdown语法，有多种多样的主题可供选择。

环境布置

安装软件

下载所需软件

  • Node.js（Node.js 版本需不低于 8.10，建议使用 Node.js 10.0 及以上版本）
  • Git

大陆地区请前往淘宝 Git for Windows 镜像、淘宝 Node.js 镜像下载

安装Git和Node.js

Git

  • Windows：下载并安装 git.

  • Mac：使用 Homebrew, MacPorts 或者下载 安装程序。

  • Linux (Ubuntu, Debian)：sudo apt-get install git-core

  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

Node.js
  • Windows：下载并安装Node.js
  • Mac：使用 Homebrew 或 MacPorts 安装。
  • Linux（DEB/RPM-based）：从 NodeSource 安装。
  • 其它：使用相应的软件包管理器进行安装，可以参考由 Node.js 提供的 指导

安装hexo

打开cmd，输入

$ node -v
$ npm -v

检查node是否安装成功（安装成功会显示当前版本号），确认无误后输入

$ npm install -g hexo-cli

选择一个你希望作为储存博客的文件夹（请使用英文名称），右键打开Git Bash Her，输入

$ hexo init  初始化文件夹

然后输入

$ npm install
$ hexo s
$ hexo g

这是打开浏览器输入网站http://localhost:4000/会进入一个网站，那么安装成功

连接到GitHub

配置ssh密钥

接上一步，Ctrl+C取向本地预览，输入

$ cd ~/. ssh  检查本地有没有ssh密钥

没有ssh密钥的话输入

$ git config --global user.name "GitHub用户名"
$ git config --global user.email "GitHub注册邮箱"
$ ssh-keygen -t rsa -C "GitHub注册邮箱"

会提示输入密码，但不建议设置密码，直接回车即可

打开C:\Users\用户名\ .ssh文件夹，用记事本打开id_rsa.pub文件，复制全部内容

登录到GitHub，选择setting–SSH and GPG keys–New SSH key

Title随意，将ssh密钥复制到Key栏

检查链接是否成功

$ ssh -T git@github.com

输入yes，回车，如果出现

Hi GitHub用户名! You've successfully authenticated, but GitHub does not provide shell access

则ssh密钥配置成功

修改_config.yml

用记事本打开博客根目录下的_config.yml

更改最后的数据（冒号后有空格！！！）

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: GitHub仓库地址（使用ssh）
  branch: master
repo获取

登录GitHub，打开在GitHub创建的博客仓库

点开文件上方绿色的Clone or download

使用ssh！！！之后点击地址栏右侧的图标复制

安装hexo-deployer-git自动部署发布工具

在本地博客文件夹根目录打开Git Bash Here，输入

npm install hexo-deployer-git  --save

发布到GitHub

在Git Bash（或cmd）中输入

$ hexo d -g

使用自定义域名

解析域名到你的博客地址

打开你的域名控制台，添加CNAME记录指向你的博客域名

在GitHub中添加你的域名

在GitHub中打开项目，选择setting，向下翻到GitHub Pages

在Custom domain中输入你的域名地址www.example.com

在本地博客文件夹添加CNAME记录

打开本地博客根目录–source创建一个文本文档

输入域名地址www.example.com

保存，将文本文档改名为CNAME（去掉.txt的后缀）

书写博客

创建新文章

在本地博客文件夹根目录打开Git Bash（或cmd）输入

$ hexo new 文章名称

编辑本地博客文件夹根目录–source–_posts–文章名称.md文件

上传

在本地博客文件夹根目录打开Git Bash（或cmd）输入

$ hexo d -g
